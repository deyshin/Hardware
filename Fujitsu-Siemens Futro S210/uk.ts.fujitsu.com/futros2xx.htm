Specifications

S2xx Common Specifications:
  • Transmeta TM5800 800 MHz processor
  • Flash Memory 32-256MB (Compact Flash Card)
  • Operating System
    • embedded Linux NG (eLuxNG) - 32/128MB CF card only
    • Windows XP Embedded (XPe) - 256MB CF card only
  • Main System Memory
    • 128-512MB
  • Integrated AC97 Audio
    • 1 x Line out (rear)
    • 1 x Microphone In
    • 1 x Headphone Jack
  • Integrated 32bit video controller
  • 4 x USB (2 x front, 2 x rear)
  • 1 x PCI midsize slot (customizable option )
  • Integrated 10/100 Mbit/s LAN Controller
  • Dimensions (HxWxD) 246x68x177mm (with stand)
  • Weight 1.5kg
  • internal power supply
Individual Model Specifications:
  • S210 (32MB CF card, 128MB memory, eLuxNG)
  • S220 (128MB CF card, 128MB memory, eLuxNG)
  • S230 (256MB CF card, 256MB memory, eLuxNG or XPe)
